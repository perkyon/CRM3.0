# Подключение домена burodigital.ru к Vercel

## Шаг 1: Добавить домен в Vercel

1. Зайдите в Vercel Dashboard → проект `crm-3-0` → **Settings** → **Domains**
2. Нажмите **"Add Domain"**
3. Введите домен: `burodigital.ru`
4. Нажмите **"Add"**

Vercel покажет вам DNS записи, которые нужно добавить.

## Шаг 2: Настроить DNS записи в reg.ru

1. Зайдите в reg.ru → **Домены** → `burodigital.ru` → **Управление**
2. Нажмите **"Изменить"** рядом с "DNS-серверы и управление зоной"
3. Выберите **"Использовать DNS-серверы reg.ru"** (если еще не выбрано)
4. Перейдите в раздел управления DNS записями

### Добавьте следующие DNS записи:

**Вариант A: Использовать A записи (если Vercel предоставит IP адреса)**

```
Type: A
Name: @
Value: [IP адрес от Vercel]
TTL: 3600
```

```
Type: A
Name: www
Value: [IP адрес от Vercel]
TTL: 3600
```

**Вариант B: Использовать CNAME записи (рекомендуется)**

```
Type: CNAME
Name: @
Value: cname.vercel-dns.com
TTL: 3600
```

```
Type: CNAME
Name: www
Value: cname.vercel-dns.com
TTL: 3600
```

**Примечание:** Vercel покажет точные DNS записи после добавления домена. Используйте те, что указаны в Vercel Dashboard.

## Шаг 3: Проверка домена

После добавления DNS записей:

1. Подождите 5-10 минут для распространения DNS
2. В Vercel Dashboard → **Domains** проверьте статус домена
3. Статус должен измениться с "Pending" на "Valid"

## Шаг 4: SSL сертификат

Vercel автоматически выдаст SSL сертификат для вашего домена после проверки DNS.

## Шаг 5: Проверка работы

После того, как домен подключен:

1. Откройте `https://burodigital.ru` в браузере
2. Должен открыться ваш CRM проект
3. Проверьте, что `https://www.burodigital.ru` тоже работает

## Если что-то не работает:

1. Проверьте DNS записи в reg.ru - они должны совпадать с теми, что указаны в Vercel
2. Подождите до 24 часов для полного распространения DNS (обычно 5-10 минут)
3. Проверьте статус домена в Vercel Dashboard
