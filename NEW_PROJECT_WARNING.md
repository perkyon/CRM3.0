# ⚠️ Внимание: Создание нового проекта в Vercel

## Последствия создания нового проекта:

Если вы создадите новый проект, вам нужно будет:

1. **Настроить все заново:**
   - Подключить репозиторий GitHub
   - Настроить домены
   - Добавить все Environment Variables
   - Настроить Build Command и Output Directory

2. **Обновить GitHub Actions:**
   - Получить новый `VERCEL_PROJECT_ID`
   - Обновить secret `VERCEL_PROJECT_ID` в GitHub

3. **Обновить домены:**
   - Если у вас есть кастомные домены, их нужно будет перенастроить

## Альтернатива (рекомендуется):

**Не создавайте новый проект!** Вместо этого:

1. Оставьте существующий проект `crm-3-0`
2. Просто сделайте новый деплой (push в `main`)
3. Новый деплой будет использовать обновленные Project Settings
4. Предупреждение исчезнет автоматически

## Если все равно хотите создать новый проект:

Убедитесь, что:
- Build Command: `npm run vercel-build`
- Output Directory: `dist`
- Root Directory: `./` (корень проекта)
- Все Environment Variables добавлены
- Репозиторий подключен правильно

Но лучше просто сделать новый деплой в существующем проекте!
